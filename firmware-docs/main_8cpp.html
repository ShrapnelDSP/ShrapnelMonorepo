void audio_event_send_callback ( const char *  message,
int  fd 
)

Called when an event is triggered.

Parameters
messageJSON representation of the audio event
fdThe socket file descriptor for the client initiating this event

fd is used to prevent a client from receiving a message that was triggered by it. The implementation of this callback must ensure that the message is transmitted only to clients where their descriptor is different from fd.
